## 📌 Overview

This repository was created as a hands-on tutorial for using **GitHub Desktop**. It has grown into a small but well-structured web project demonstrating:

- HTML/CSS/JS project structure
- State management patterns
- API service layers
- Design systems with CSS variables
- Git best practices

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- [GitHub Desktop](https://desktop.github.com/)
- (Optional) [VS Code](https://code.visualstudio.com/)

### Run Locally

```bash
git clone https://github.com/nikhiltomar2712/desktop-tutorial.git
cd desktop-tutorial
open index.html
```
